Theme Dictionary is a practical theme recolor tool for Blender.  
Instead of manually digging through long theme preference trees, it helps you **pick a color, locate related theme entries, and edit them from a focused results list**.

---

## Overview

Theme Dictionary is built for artists, UI designers, and anyone who customizes Blender themes frequently.

It turns Blender's theme settings into a more visual workflow:

- sample a UI color with an eyedropper
- search theme entries by similar color
- refine the result range with tolerance
- jump to relevant items more quickly
- keep track of recently modified entries

---

## Key Features

### 1. Eyedropper-based color lookup
Pick a color directly from Blender's interface and let the add-on retrieve theme entries that match or closely resemble it.

### 2. Adjustable color tolerance
Use a tolerance value to control how strict the search should be. This makes it easier to find either exact matches or a wider set of visually related colors.

### 3. Direct access to relevant theme entries
Instead of browsing through many nested theme categories, you get a filtered list of relevant options that can be inspected and edited more efficiently.

### 4. Linked Item / Common Properties split
In area detect mode, search results are separated into:
- **Linked Item**: options more specifically tied to the current area
- **Common Properties**: options shared more broadly across the UI

This helps surface the most useful items first.

### 5. Edit history mode
Modified items are collected into a history-style list, so you can quickly revisit what you changed.

### 6. Character search
A bottom search box lets you find related entries by name. It supports both **Chinese and English** search terms.

### 7. Chinese translation support
The add-on includes Chinese translation support for a friendlier experience in Blender's Chinese interface.

## Installation

### Install from Blender Extensions
1. Open **Blender**.
2. Go to **Edit → Preferences → Get Extensions**.
3. Search for **Theme Dictionary**.
4. Install and enable it.

### Install from GitHub Release
1. Open the latest release page: [https://github.com/VODIAC/BlenderAddon_theme_dictionary/releases/latest](https://github.com/VODIAC/BlenderAddon_theme_dictionary/releases/latest)
2. Download the packaged extension/add-on file.
3. In Blender, go to **Edit → Preferences → Add-ons**.
4. Click **Install from Disk**.
5. Select the downloaded file and enable **Theme Dictionary**.
